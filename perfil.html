Ramais

Uma lista de Ramais

Frontend — Backend

Esse projeto é legal pelo fato que ele é extremamente util, Aqui na prefeitur ainda usamos sistemas de PABX, porém o principal problema é que funcionario mudavam de setor e junto muitas vezes requisitavam a permanencia dos ramais.

Minha solução foi essa, é um projeto usando Django-Python e Bootstrap, consigo entregar um lugar unico para consulta dos ramais, como alterar os numeros e as pessoas.

Pra rodar é só criar um bash script :

" python3 manage.py runserver IP:PORT "

||

This project is cool for the fact that it is extremely useful, Here in the city hall we still use PBX systems, but the main problem was that employees changed sectors and often requested the permanence of the extensions.

My solution was this, it is a project using Django-Python and Bootstrap, I can deliver a unique place to query the extensions, how to change the numbers and people.

To run it just create a bash script :

" python3 manage.py runserver IP:PORT "
